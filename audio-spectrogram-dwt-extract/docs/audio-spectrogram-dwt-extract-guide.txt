LAB 6 - TACH ANH TU AUDIO STEGO BANG DWT VA KEY

Muc tieu
-------
Sender co san hai file:

  1. stego.wav  - audio da duoc nhung anh bi mat
  2. secret.key - key va thong tin can thiet de tach anh

Receiver se lam tung cong doan:

  1. Nhan stego.wav va secret.key.
  2. Tach tin hieu bi mat tu he so tan so cao cua DWT.
  3. Dung key de dao hoan vi va dung lai anh.
  4. Mo anh da tach de quan sat.

Sinh vien chi can kiem tra lenh cua tung task tao dung file dau ra. Khong can chay checkwork sau tung task; checkwork duoc chay mot lan o cuoi bai.

Task 1 - Kiem tra file o sender
-------------------------------
Mo terminal sender:

  cd ~/stego
  ls -l
  cat README_sender.txt
  ./play_stego.sh

Sender phai co:

  stego.wav
  secret.key

Task 2 - Bat SSH tren receiver
------------------------------
Mo terminal receiver:

  sudo service ssh start
  systemctl status ssh

Task 3 - Gui stego.wav va secret.key sang receiver
--------------------------------------------------
Mo terminal sender:

  scp ~/stego/stego.wav ~/stego/secret.key ubuntu@receiver:~/stego/
  ssh ubuntu@receiver "python3 ~/stego/refresh_status.py"

Mo terminal receiver va kiem tra:

  ls -l ~/stego/stego.wav ~/stego/secret.key

Task 4 - Tach tin hieu bi mat tu DWT
------------------------------------
Mo terminal receiver:

  cd ~/stego
  nano extract_signal.py

Sua TODO thanh:

  STEGO_FILE = "stego.wav"
  KEY_FILE = "secret.key"

Chay:

  python3 extract_signal.py
  ls -l hidden_signal.json

Task 5 - Dung key de khoi phuc anh
----------------------------------
Mo terminal receiver:

  nano recover_image.py

Sua TODO thanh:

  HIDDEN_SIGNAL = "hidden_signal.json"
  KEY_FILE = "secret.key"

Chay:

  python3 recover_image.py
  ls -l recovered_secret.png

Task 6 - Mo anh da tach
-----------------------
Trong receiver:

  ./view_recovered.sh

  checkwork
